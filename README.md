# skigame

A lightweight cross-country skiing arcade game built with plain HTML5 Canvas.

## Play Online

After GitHub Pages is enabled for this repo:

https://phfilip.com/skigame

## Goal

Ski as long as possible, avoid obstacles, and maximize score.

## Controls

- Move up lane: `A` or `Left Arrow`
- Move down lane: `D` or `Right Arrow`
- Go fast: `W`, `Up Arrow`, or `Space`
- Brake: `S` or `Down Arrow`
- Restart after crash: `R`
- Toggle help panel: `H`
- Close help panel: `Esc`

## Scoring

- Score increases continuously while you move.
- Going faster increases score gain proportionally.
- Collecting berries adds bonus points.
